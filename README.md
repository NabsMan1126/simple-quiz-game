# simple-quiz-game
A Python CLI multiple-choice quiz game that tracks the user’s score.
## Description

This is a simple command-line Quiz Game built with Python. The program presents multiple-choice questions to the user, evaluates their answers, and keeps track of the score throughout the game.

At the end of the quiz, the final score is displayed based on the number of correct answers. This project helps practice working with data structures like lists and dictionaries, along with loops, conditional statements, and basic scoring logic.
